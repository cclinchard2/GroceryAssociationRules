# Association Rule Mining - Groceries
Association Rule Mining with Grocery Dataset


### Dataset Description:
- 38,765 rows of data about grocery store purchases

### High-Level Summary

- Purchasing frankfurters increased the chances of purchasing other vegetables
- Most rules had whole milk in the association, but these associations were weaker than random chance (i.e., lift < 1)
